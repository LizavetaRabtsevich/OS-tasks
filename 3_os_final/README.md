Лабораторная работа №3
**Тема: «Синхронизация потоков при помощи критических секций и событий. Обработка тупиков»**

Описание программы
Многопоточное приложение для работы с общим массивом данных, демонстрирующее механизмы синхронизации и обработку тупиковых ситуаций.

Структура потоков:
Поток main - управляющий поток
Потоки marker - рабочие потоки (несколько экземпляров)

Функциональность потоков:
Поток main:
Выделение и инициализация массива целых чисел (нулями)
Запуск заданного количества потоков marker
Управление работой потоков через сигналы
Отслеживание состояния системы и обработка тупиков
Визуализация состояния массива

Поток marker:
Генерация случайных чисел для выбора индексов массива
Маркировка элементов массива (присвоение своего номера)
Обнаружение конфликтных ситуаций (элемент уже помечен)
Обработка сигналов от main (продолжить/завершить)
Очистка помеченных элементов при завершении

Ключевые особенности:
Использование критических секций для защиты общего массива
Сигнальные события для межпоточной коммуникации
Механизм обнаружения и разрешения тупиковых ситуаций

Визуальный вывод состояния системы

Контролируемое завершение потоков
