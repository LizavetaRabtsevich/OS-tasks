Лабораторная работа №5
Тема: «Обмен данными по именованным каналам»

Система управления доступом к файлу данных через именованные каналы с поддержкой многопользовательского доступа.

Архитектура системы
Процесс-Сервер:
Создание бинарного файла с данными сотрудников
Запуск процессов-клиентов
Обслуживание запросов через именованные каналы
Управление блокировками записей
Контроль доступа на чтение/запись

Процесс-Клиент:
Подключение к серверу через именованный канал
Запросы на модификацию и чтение записей
Работа с записями по ID сотрудника
Консольный интерфейс управления

Механизмы доступа
Блокировки записей:
Чтение: множественный доступ разрешен
Запись: эксклюзивный доступ с блокировкой
Гибрид: чтение блокирует запись, но не чтение
