Лабораторная работа №1
Тема: «Создание процессов»

Система для работы с данными сотрудников, состоящая из трех программ:

Главная программа (Main):

Запрашивает параметры с консоли

Запускает утилиты Creator и Reporter как отдельные процессы

Ожидает завершения процессов с помощью WaitForSingleObject

Выводит результаты на экран

Утилита Creator:

Создает бинарный файл с данными сотрудников

Получает имя файла и количество записей через командную строку

Данные вводятся с консоли вручную

Утилита Reporter:

Создает текстовый отчет на основе бинарного файла

Рассчитывает зарплату для каждого сотрудника

Формирует таблицу с данными и расчетами

Особенности реализации:
Использование Windows API для работы с процессами

Передача параметров через командную строку

Синхронизация процессов через WaitForSingleObject

Работа с бинарными и текстовыми файлами
